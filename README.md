# 端口扫描器 (Port Scanner)

使用 **Go 语言** 编写的高效端口扫描器，支持 IP 段扫描和端口服务探测。

## 功能 (Features)
- 🔍 **支持 IP 范围扫描**（CIDR、范围、通配符格式）
- ⚡ **并发扫描**，提高扫描效率
- 🔎 **识别常见服务**（如 FTP、HTTP、Redis 等）
- 🛠 **自定义端口范围**，适应不同需求
